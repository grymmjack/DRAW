# TOOLBOX and OPTIONS/STATUS

## MOVE 

### TOOLTIP
m - MOVE - Position current selection or selected layers on the canvas.

### OPTIONS
(o) Current Layer
( ) Picked Layer

### STATUS
DRAG move. ALT clones. SHIFT constrains. ARROWS nudge. SHIFT+ARROWS = nudge more.


## PAN

### TOOLTIP
SPACE - PAN - Pan the canvas in the window to adjust what is in view.

### STATUS
DRAG pan. D-CLICK center / fit.


## ZOOM

### TOOLTIP
z - ZOOM - Magnify part of the canvas for more precise drawing.

### OPTIONS
[ 50% ] [ 100% ]  [200% ] [ 400% ] [ FIT ]

### STATUS
DRAG zoom area. M-WHEEL zoom in/out. 


## CROP

### TOOLTIP

CROP - Select a part of the canvas and trim everything outside away.

### STATUS
DRAG select. SHIFT constrain. 


## RECTANGLE MARQUEE

### TOOLTIP
m - RECTANGLE MARQUEE - Create rectangular selections.

### STATUS
DRAG to select. SHIFT add. ALT subtract.


## FREEHAND MARQUEE

### TOOLTIP
FREEHAND MARQUEE - Create any shape free-form selections.

### STATUS
DRAW to select. SHIFT+CTRL Snap. SHIFT add. ALT subtract.


## POLYGONAL MARQUEE

### TOOLTIP
POLYGONAL MARQUEE - Create a selection shape using points and lines.

### STATUS
DRAW selection shape. L-CLICK create point. R-CLICK finish. 


## ELLIPTICAL MARQUEE

### TOOLTIP
ELLIPTICAL MARQUEE - Create an elliptical shaped selection.

### STATUS
DRAG to select. SHIFT add. ALT subtract.


## MAGIC WAND MARQUEE

### TOOLTIP
w - WAND MARQUEE - Create a selection by same colors.

### STATUS
CLICK to select on layer. SHIFT+CLICK to select on full image. SHIFT add. ALT subtract.


## COLOR PICKER

### TOOLTIP
i - COLOR PICKER - Use the color syringe to pickup a color off the canvas.

### STATUS
L-CLICK to pick FG color. R-CLICK pick BG color. HOLD R sample reference.


## TEXT
[FONT PICKER ^] [SIZE ^] [LOAD] [DONE]

### TOOLTIP
t / SHIFT+t / CTRL+t - Create text / small text / custom font text.

### STATUS
CLICK position cursor. TYPE to enter text. ESC done.


## ERASER

### TOOLTIP
e - ERASER - Erase pixels back to transparency.

### STATUS
DRAG erase. SHIFT+DRAG erase all layers. HOLD E in other tools to temp. erase.


## DOT

### TOOLTIP
d - DOT - Paint once at brush size, or place a single custom brush.

### STATUS
CLICK draw. D-CLICK reset brush. L-CLICK FG color. R-CLICK BG color. HOLD E erase.


## BRUSH

### TOOLTIP
b - BRUSH - Paint continuously.

### STATUS
DRAG draw. L-CLICK FG color. R-CLICK BG color. HOLD E erase. CTRL+WHEEL brush size.


## SPRAY

### TOOLTIP
k - SPRAY - Paint with randomly distributed points at brush size.

### STATUS
DRAG draw. L-CLICK FG color. R-CLICK BG color. HOLD E erase. CTRL+WHEEL spray size.


## FILL

### TOOLTIP
f - FILL - Color in contiguous areas of canvas with borders or same colors.

### STATUS
L-CLICK fill FG color. R-CLICK fill BG color. SHIFT+CLICK canvas area.


## LINE

### TOOLTIP
l - LINE - Create straight lines at any angle. 

### STATUS
DRAG draw. L-CLICK FG color. R-CLICK BG color. CTRL+SHIFT snap.


## LINE POLYGON

### TOOLTIP
p - POLYGONAL LINE - Create polygonal shapes using connected lines.

### STATUS
DRAW shape. L-CLICK create point. R-CLICK finish. 


## FILLED POLYGON

### TOOLTIP
SHIFT+p - FILLED POLYGONAL SHAPE - Create filled polygonal shapes using connected lines.

### STATUS
DRAW filled shape. L-CLICK create point. R-CLICK finish. 


## LINE RECTANGLE

### TOOLTIP
r - RECTANGLE - Create rectangle shapes on the canvas.

### STATUS
DRAW rectangle. L-CLICK FG color. R-CLICK BG color. CTRL+SHIFT snap.


## FILLED RECTANGLE

### TOOLTIP
SHIFT+r - FILLED RECTANGLE - Create filled rectangle shapes on the canvas.

### STATUS
DRAW filled rectangle. L-CLICK FG color. R-CLICK BG color. CTRL+SHIFT snap.


## ELLIPSE

### TOOLTIP
c - ELLIPSE/CIRCLE - Create ellipses and circles on the canvas.

### STATUS
DRAW ellipse. L-CLICK FG color. R-CLICK BG color. CTRL+SHIFT snap.


## FILLED ELLIPSE

### TOOLTIP
SHIFT+c - FILLED ELLIPSE/CIRCLE - Create filled ellipses and filled circles on the canvas.

### STATUS
DRAW filled ellipse. L-CLICK FG color. R-CLICK BG color. CTRL+SHIFT snap.


## EXPORT SELECTION

### TOOLTIP
CTRL+SHIFT+ALT+s - EXPORT SELECTION.

### STATUS
Saves the selected area to disk.


## SAVE IMAGE

### TOOLTIP
CTRL+s - SAVE IMAGE to disk.

### STATUS
CTRL+SHIFT+s SAVE AS NEW image to disk.


## LOAD IMAGE

### TOOLTIP
CTRL-O - LOAD IMAGE from disk.

### STATUS
SHIFT+CTRL+o - IMPORT and PLACE image from disk.


## EXPORT CODE

### TOOLTIP
EXPORT QB64PE Source Code.

### STATUS
Save as .BAS for learning and dissection and usage in your own programs.


## COMMAND PALETTE

### TOOLTIP
? - COMMAND PALETTE.

### STATUS
Show a searchable cheat sheet of commands and keyboard shortcuts.
