# [ ] SYMMETRY TESTING

## [ ] MODE CYCLING

### [ ] Cycle Symmetry Modes
#### [ ] F7 cycles through modes
1. [ ] Press `F7` — verify Vertical symmetry (|) enabled
2. [ ] Press `F7` — verify Cross symmetry (+) enabled
3. [ ] Press `F7` — verify Asterisk symmetry (*) enabled
4. [ ] Press `F7` — verify symmetry Off

---

## [ ] SYMMETRY DRAWING

### [ ] Vertical Symmetry (|)
#### [ ] Mirror left/right
1. [ ] Enable Vertical symmetry
2. [ ] Draw on one side of center axis
3. [ ] Verify mirrored copy appears on opposite side
4. [ ] Verify 2 copies total (original + mirror)

### [ ] Cross Symmetry (+)
#### [ ] Mirror H+V (4 copies)
1. [ ] Enable Cross symmetry
2. [ ] Draw in one quadrant
3. [ ] Verify all 4 quadrants have mirrored content

### [ ] Asterisk Symmetry (*)
#### [ ] Mirror 8-way
1. [ ] Enable Asterisk symmetry
2. [ ] Draw a stroke
3. [ ] Verify 8 mirrored strokes appear

---

## [ ] SYMMETRY CENTER

### [ ] Reposition Center
#### [ ] Ctrl+Left-Click to reposition
1. [ ] Enable symmetry
2. [ ] Ctrl+Left-Click on canvas at a new position
3. [ ] Verify symmetry center moves to clicked position
4. [ ] Draw — verify mirroring uses new center

---

## [ ] SYMMETRY OFF

### [ ] Disable Symmetry
#### [ ] F8 turns off symmetry (when no custom brush)
1. [ ] Enable any symmetry mode
2. [ ] Press `F8` (no custom brush/paint mode active)
3. [ ] Verify symmetry turns off
