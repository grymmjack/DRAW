# [ ] ZOOM TESTING

## [ ] TOOL ACTIVATION

### [ ] Activate Zoom Tool
#### [ ] Activate via Z key
1. [ ] Press `Z`
2. [ ] Verify toolbar highlights Zoom tool

---

## [ ] ZOOM IN / OUT

### [ ] Zoom Controls
#### [ ] Ctrl+= zooms in
1. [ ] Press `Ctrl+=`
2. [ ] Verify canvas magnification increases
3. [ ] Verify zoom percentage updates in status bar

#### [ ] Ctrl+- zooms out
1. [ ] Press `Ctrl+-`
2. [ ] Verify canvas magnification decreases

#### [ ] Mouse wheel zooms
1. [ ] Scroll wheel up — verify zoom in
2. [ ] Scroll wheel down — verify zoom out

#### [ ] Zoom tool left-click zooms in
1. [ ] Activate Zoom tool
2. [ ] Left-click on canvas
3. [ ] Verify zoom in centered on click position

#### [ ] Zoom tool right-click zooms out
1. [ ] Right-click with Zoom tool
2. [ ] Verify zoom out

---

## [ ] ZOOM PRESETS

### [ ] Fit and Reset
#### [ ] Ctrl+0 fits canvas
1. [ ] Press `Ctrl+0`
2. [ ] Verify canvas fits to window

#### [ ] Ctrl+1 resets to 100%
1. [ ] Press `Ctrl+1` (if bound)
2. [ ] Verify zoom resets to 100%

---

## [ ] PIXEL GRID

### [ ] High-Zoom Pixel Grid
#### [ ] Pixel grid appears at 400%+
1. [ ] Zoom to 400% or higher
2. [ ] Enable pixel grid (`Shift+'`)
3. [ ] Verify individual pixels have grid lines between them
