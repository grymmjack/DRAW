# [ ] CROSSHAIR TESTING

## [ ] ACTIVATION

### [ ] Toggle Crosshair
#### [ ] Shift or Caps-Lock activates
1. [ ] Hold `Shift` while using a drawing tool
2. [ ] Verify crosshair guide lines appear (horizontal + vertical through cursor)
3. [ ] Release `Shift`
4. [ ] Verify crosshair disappears

#### [ ] Caps-Lock for persistent crosshair
1. [ ] Enable `Caps-Lock`
2. [ ] Verify crosshair stays visible
3. [ ] Move mouse — verify crosshair follows
4. [ ] Disable `Caps-Lock`
5. [ ] Verify crosshair goes away
