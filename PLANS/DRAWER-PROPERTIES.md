# DRAWER

## PATTERN MODE

### TOOLTIP
F3 - DRAW MODE: PATTERN 

### STATUS
All fills will use patterns. Pick color to change opaque pixels of pattern. R-CLICK on patterns for menu. 

## GRADIENT MODE
F2 - DRAW MODE: GRADIENT

### TOOLTIP
All fills will use gradients. 

### STATUS
R-CLICK on gradients for menu, and editor.


## BRUSH MODE
F1 - DRAW MODE: BRUSH - x/L-CLICK BL = SWAP FG/BG. R-CLICK BL = DEFAULT colors.

### TOOLTIP
All drawing tools will use custom brushes. R-CLICK on brushes for menu.


## BRUSH SIZE

### TOOLTIP
CTRL+WHEEL - Change brush size. Click width to change.

### STATUS
Make selection and use CTRL-b to create custom brush, then store it in bin for later recall.


## GRID SNAP

### TOOLTIP
; - SNAP TO GRID - Toggle grid snapping. WHEEL over icon = change SNAP TYPE.

### STATUS
/ to change SNAP TYPE - CENTER, or CORNER.


## GRID

### TOOLTIP
' - GRID - Toggle show/hide of grid. WHEEL over icon to change GRID TYPE.

### STATUS
. / , to change GRID size. SHIFT+' show/hide PIXEL GRID. CTRL+SHIFT+/ make GRID brush size.


## SYMMETRY

### TOOLTIP
SYMMETRY MODE - L-CLICK toggle. WHEEL over icon change SYMMETRY TYPE.

### STATUS
DRAW mirrored pr kaleidoscopic patterns at once.


## PALETTE OPS

### TOOLTIP
PALETTE OPS - L-CLICK remap image to current palette.

### STATUS
Try CTRL-ALT-r to choose a RANDOM palette then remap, for inspiration!


## COLOR OPS

### TOOLTIP
COLOR OPS - L-CLICK to IMPORT, R-CLICK to LOAD from Lospec

### STATUS
See the PALETTE MENU in the MENUBAR for more options.


## TRANSFORM OPS

### TOOLTIP
TRANSFORM OPS - L-CLICK icon on left to flip HORIZONTALLY. right to flip VERTICALLY.

### STATUS
See the EDIT -> TRANSFORM menu in the MENUBAR for more transformations.


## CANVAS OPS

### TOOLTIP
CANVAS OPS - L-CLICK to switch to SPARE PAGE. R-CLICK to CLEAR SPARE PAGE.

### STATUS
Use SPARE PAGE as a scratchpad, test area, assembly canvas, etc.


## MINI PALETTE

### TOOLTIP
MINI PALETTE - L-CLICK choose FG color. R-CLICK choose BG color.

### STATUS
You can use any image for this palette, see the documentation.
