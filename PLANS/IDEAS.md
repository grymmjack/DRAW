# IDEAS

## TEXT FORMATTING

### ATTRIBUTES

#### Per text entry:

- Font
- Font size
- Line Height
- Baseline
- Kerning
  - Character spacing between each (custom tuning)
- [x] Font Hinting
- [x] Monospace

#### Per selected character (or all if none selected):

- FG Color
- BG Color (inc. transparent)
- BOLD
- Italic
- Underline
- Strikethrough
